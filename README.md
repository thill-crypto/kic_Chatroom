## Chatroom for a Rasa Chatbot

🇬🇧 **English Version**
#### Take a look on the chatroom here:  
[https://thill-crypto.github.io/kic_chatroom.github.io](https://thill-crypto.github.io/kic_chatroom.github.io)

#### To create your own chatroom:
1. Fork this repository.
2. Edit `index.html` and replace the host URL with your own chatbot URL.
3. Click **Commit changes**.
4. Activate GitHub Pages in Settings.
